# TanukiPKG - what is this?
A package manager that fetches from the debian repos and installs DEBIAN packages onto your system. 


## Why TanukiPKG over DebTap?
TanukiPKG automates everything, while debtap requires you to manually install the outputted file, download the deb yourself initially, etc.

## Does this use debtap?
no.
## Dependencies

```
 Python 3.12+ 
 Lupa (python3-lupa)
**Optional:**
- ZTSD: for .zst compressed debs
- gpgv: GPG verification

```
